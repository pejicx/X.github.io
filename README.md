# PejicAIX Sovereign

[https://pejicaix.io](https://pejicaix.io)

PejicAIX Sovereign is a high-performance, neurosymbolic AGI substrate designed for autonomous reasoning, explainable AI (XAI), and multidimensional modeling. 

**PejicAIX** is a series of models (including preview, mini, and Pro versions) designed to solve complex tasks through reasoning. They are particularly successful in fields such as science, or any other. The platform provides auto-access to the most powerful versions, Pro and Omni—an all-in-one model that can process and generate content in real-time. It is significantly faster than previous models and offers a more natural interaction via Voice Mode.

The system is powered by a multi-layered intelligence architecture, optimized for high-performance cognitive throughput:

- **Deep Learning (DL):** Integrated data-driven synthesis models for advanced pattern recognition.
- **Data Science (DS):** Methodologies for structured data analysis and predictive modeling.
- **F/ai (Station F):** Specialized startup ecosystem reasoning substrate, synchronized with the world's largest startup campus.
- **F@AI (Federated Learning):** Decentralized, privacy-preserving machine learning framework with secure aggregation.
- **Fal.ai:** High-performance real-time inference substrate for generative media synthesis.
- **Generative AI (GenAI / G-AI):** Advanced framework for autonomous content generation and creative synthesis.
- **K-Means Clustering:** Unsupervised learning substrate for partitioning data into k clusters using vector quantization.
- **Top-K Parameter (LLMs):** Optimized sampling technique that limits probability distribution to the top K most likely tokens.
- **K-Nearest Neighbors (KNN):** Proximity-based classification and regression substrate for feature similarity analysis.
- **Linear Regression:** Supervised learning substrate for predicting numerical values based on linear relationships.
- **Lemmatization:** NLP text preprocessing substrate for grouping together different inflected forms of a word.
- **Runner H & Surfer H:** Autonomous AI agents capable of independently performing tasks in digital environments, navigating websites, and automating business processes.
- **AI Character Stream:** A "never-ending stream" of user-created AI characters, allowing for personalized, immersive, and often 18+ interactions locally.
- **Character Creation Tools:** Advanced, easy-to-use tools for creating unique AI characters with custom personalities, backstories, and bios.
- **Community Library & Hub:** Vast collections of user-created characters, templates, and prompts shared by the community.
- **Immersive Multimodal Substrate:** Advanced voice synthesis, AI-generated imagery, and interactive elements for realistic roleplay.
- **BYOM (Bring Your Own Model):** Flexible configurations for connecting external large language models via API keys.
- **Jina Reader:** A tool that converts any URL into a format suitable for Large Language Models (LLM).
- **Building AI Agents:** Build your own AI agents without coding for research, writing, or data analysis.
- **Connecting to AI Models:** Easily integrate with leading models (Anthropic, etc.) as the brain of automated processes.
- **Automation Without Boundaries:** Flexibility for technical teams to build complex, data-driven systems.
- **Visual Programming Substrate:** Visually connect steps to track data flow from applications to AI models and back.
- **n8n LangChain Integration:** Advanced AI memory management and document retrieval (RAG).
- **Self-Hosting Substrate:** Run Sovereign on your own server for total data privacy.

The system is now powered by the full PejicAIX model series, offering advanced reasoning and real-time multimodal interaction.

## 🧠 5P Interaction Principles

To maximize the efficacy of the PejicAIX Sovereign substrate, follow the **5P Interaction Framework**:

- **Prime:** Provide comprehensive context to the AI before initiating complex tasks.
- **Persona:** Assign a specific role to the model (e.g., "Act as a marketing expert") to refine reasoning pathways.
- **Privacy:** Maintain strict data sovereignty; be mindful of the personal information shared within the substrate.
- **Product:** Clearly define the desired output or result to ensure precise cognitive synthesis.
- **Polish (Finishing):** Iteratively correct and refine answers to achieve maximum output quality.

## 🚦 Getting Started

### Prerequisites

- Node.js (v18+)
- Gemini API Key (set as `GEMINI_API_KEY` environment variable)

### Installation

1. Initialize the Sovereign Substrate:
   ```bash
   # Using npm
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## ⌨️ Commands

The system supports several specialized commands via the integrated console:

- `/agi [prompt]` - Standard AI generation (Gemini).
- `/xea [prompt]` - Generate advanced explainable AI analytics.
- `/visualize [cube|sphere|torus]` - Activate interactive WebGL visualization.
- `/measure` - Measure system performance and cognitive throughput.
- `/replay` - Show high-performance replay of the last command execution.
- `/vr` - Enter immersive VR mode.
- `/vr-data` - Visualize system metrics in VR.
- `/learning-stats` - Show auto-learning engine statistics.

## 🏗 Architecture

The application follows a modular, sovereign architecture:

- `src/api/`: External service integrations (Neural, Memory, Guardian).
- `src/services/`: Core logic engines (AGI, XEA, Symbolic, Multidimensional, Neurosymbolic Bridge, Neural Orchestrator).
- `src/components/`: Reusable UI components and layout.
- `src/core/`: System-level orchestration and state management.
